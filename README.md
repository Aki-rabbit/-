# Trip Phrase Cards (仮)

## Overview
海外旅行で使える「ちょっとした一言」を、現地の言葉でさっと確認できるフレーズカードアプリ。

## Purpose
海外旅行中に、挨拶や短い単語を使えるようになることで現地の人とのコミュニケーションを活性化させる。  
言語の壁による不安を少しでも減らし、気持ちのいいやりとりを生み出すことを目的とする。

## Core Features (MVP)
- オフラインでも使えるよう、手元に保存して即時表示できる
- シーン（例：レストラン）ごとに単語・フレーズカードを管理
- カードはタップで表裏が切り替わる（イタリア語 ↔ 日本語）
- 最小構成：レストランで使う「ありがとう（Grazie）」カード

## Future Enhancements
- 同行者とリンク1つで共有できる共有機能  
- 複数言語の切り替え機能  
- 他言語との比較モード（例：イタリア語＋英語＋スペイン語）  
- お気に入り / 履歴機能で“旅中の辞書”として使いやすく  
- PWA 化して完全オフライン対応  
- Rails バックエンドで「旅セット」を保存しURL化
